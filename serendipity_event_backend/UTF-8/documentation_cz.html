Dokumentace k pluginu 'Backend URL'
(serendipity_event_backend)

Dokumentaci k tomuto pluginu přeložil do češtiny Vladimír Ajgl (vlada [zavinac] ajgl [tecka] cz) a naposledy ji aktualizoval dne 22.1.2012. Od té doby mohl být plugin pozměněn nebo mohly být rozšířené jeho funkce. Zkontrolujte pro jistotu i aktuální anglické "readme".

Javascriptový backend pro zobrazování příspěvků na webových stránkách mimo Váš blog

Použití

Jednoduše přidejte "<script src=[URL]></script>" na Vaší externí webovou stránku přesně v místě, kde se má zobrazit příspěvek.

Můžete také použít CSS styl pro změnu vzhledu těchto zobrazených příspěvků. Můžete použít tři css-třídy. Prohlédněte si prosím soubor "backend.html" v tomto adresáři, který obsahuje názorný příklad.

Poznámka: Můžete přidat tolik tagů "<script src=[URL]></script>", kolik chcete. Každý může mít jiné volby a nastavení.

[URL] má tvar přibližně takovýto:
http://vas.blog.cz/plugin/[BACKEND_URL]?[VOLBY]
Kde [BACKEND_URL] je název, který jste zadali v nastavení pluginu v administrační sekci.

[VOLBY] mohou být cokoliv z následujícího:

Volby

Je několik voleb, které můžete přidat k url adrese skriptu. Ovlivníte jimi vzhled zobrazených příspěvků.
název volbypřípustné hodnotyvysvětlivky
details1|0zobrazuje buď pouze nadpis nebo celý příspěvek (bez rošířené textové části!)
category[název kategorie]Pokud je nastaveno, budou se zobrazovat pouze příspěvky ze zadané kategorie. Pokud není uvedeno, budou zobrazeny příspěvky ze všech kategorií.
categoryid[N]Výpis můžete omezit na kategorie také podle jejího identifikačního čísla.
authorid[N]Nastavte ID číslo autora pro výpis pouze jeho příspěvků.
num[N]Pokud je nastaveno, bude se zobrazovat [N] příspěvků. Pokud není zadáno, zobrazuje se 10.
orderASC|DESCřazení příspěvků (podel data, ASC = vzestupně, DESC = sestupně)
date1|0Pokud je nastaveno 1, bude se zobrazovat datum vytvoření příspěvků.
dateformat[formatstring]Zformátuje zobrazované datum příspěvku podle zadaného formátovacího řetězce (přípustné je cokoliv, co umožňuje php funkce date()).
time1|0Pokud je nastaveno 1, bude se zobrazovat čas vytvoření příspěvků.
timeformat[formatstring]Zformátuje zobrazovaný čas příspěvku podle zadaného formátovacího řetězce (přípustné je cokoliv, co umožňuje php funkce date()).
point[řetězcová konstanta, text]Použije tento text jako první část nadpisů. Může být použito pro vytvoření seznamu (přidání tagů <li> nebo něco podobného)

Příklady

Následující příklad zobrazí až 30 nadpisů s date a časem vytvoření, každé s odrážkou "*" a pouze z kategorie "s9y"
<script src="http://blog.linux-stats.org/plugin/backend?category=s9y&num=30&order=DESC&date=1&dateformat=d.m.Y&time=1&timeformat=H:i&point=*"></script>

Následující příklad zobrazí kompletní příspěvky, tak jak jsou zadané na původních stránkách.
<script src="http://blog.linux-stats.org/plugin/backend?details=1&category=s9y&num=30&order=DESC&date=1&dateformat=d.m.Y&time=1&timeformat=H:i"></script>
